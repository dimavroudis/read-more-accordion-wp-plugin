A simple and modern widget to create accorrdion style "Read More" section. Fully customizable.

== Description ==

As simple as it gets! A widget usefull for using in your sidebar or page builder, like SiteOrigin, to show and hide text. 

**Feautures**

* Customizable "read more"/ "read less" text.
* Basic HTML tags allowed in text.
* Modern & minimal design. 
 

== Installation ==

**From your WordPress dashboard**
1. Visit 'Plugins > Add New'
1. Search for 'Read More Accordion Widget'
1. Activate Read More Accordion Widget from your Plugins page.

**From WordPress.org**
1. Download Read More Accordion Widget
1. Upload the 'read-more-accordion' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
1. Activate Read More Accordion Widget from your Plugins page. 

**Once Activated**
1. Visit 'Appearance > Widgets' (or if you are using SiteOrigin, visit the page you want to add the widget, click to edit, and from editor's toolbar click "Add Widget")
1. Select the widget "Read More Accordion" and added to the sidebar you want.
1. Fill the widget with you content and click "Save".
